Projects

Crypto Currency Price Prediction

In this study, I especially focused on a popular cryptocurrency, i.e., bitcoin. From many types of virtual currencies, bitcoin has a great acceptance by different bodies such as investors, researchers, traders, and policymakers. To the best of our knowledge, our target is to implement the efficient Machine learning-based prediction models specifically long short-term memory (LSTM) and gated recurrent unit (GRU) to handle the price volatility of bitcoin and to obtain high accuracy. Our study involves comparing these two-time series Machine learning techniques and proved the efficacy in forecasting the price of bitcoin.

Image Colorization Using ML Model

This project involves an image-to-image transformation using deep learning. It converts RGB images into the Lab color space, separates the L channel as input, and trains a neural network to predict the ab channels. These channels are then recombined to create Lab images, which are finally converted back to RGB. The result is an innovative approach to add color information to grayscale images.
